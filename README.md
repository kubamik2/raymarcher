# Raymarcher
Prosty edytor wykorzystujący technikę renderowania "raymarching". Pozwala na dodawanie ustalonych brył i wykonywanie operacji na zbiorach.

## Uruchomienie
`cmake -S . -B build` \
`cmake --build build && ./build/raymarcher`

## Biblioteki
- GLFW
- Dear ImGui
- stb_image
- GLAD
- OpenGL
